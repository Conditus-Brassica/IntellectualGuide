Основа для реализации шаблона Command.

Сендер отправляет команду. Команда знает агента, который должен выполнить действия.
Модуль асинхронный, вызовы делаются через await.
Агенты, которые могут отправлять команду наследуются от Sender.
Любая команда наследуется от BaseCommand. Если целевая функция агента принимает аргументы, то перегружается конструктор
команды. Базовый конструктор принимает ссылку на целевого агента. Обязательно требуется перегрузка метода execute()
класса BaseCommand. В этом методе вызывается целевой метод агента. В идеале должна быть фабрика команд, которая знает
целевого агента команды и знает, что требуется в качетсве аргументов. Фабрика собирает команды.
